🌍 37 Amharic Keywords

Develop real Python programs using full Amharic syntax. Every standard control flow and built-in type is supported natively without performance loss.

🔌 Plugin Architecture

Our compiler architecture permits adding any Ethiopian language in one simple JSON file. Contribute to Oromo, Tigrinya, and more.

⚡ Zero Overhead

Because AmharicScript operates as a source-to-source transpiler parsing into AST, the generated Python runs at native CPython speed.

Amharic Source (.amh)
ክፍለ ተማሪ:
    ስራ __init__(እኔ, ስም):
        እኔ.ስም = ስም

    ስራ አሳይ_ስም(እኔ):
        አሳይ(f'የተማሪው ስም {እኔ.ስም} ነው')
Generated Python (.py)
class ተማሪ:
    def __init__(self, ስም):
        self.ስም = ስም

    def አሳይ_ስም(self):
        print(f'የተማሪው ስም {self.ስም} ነው')
