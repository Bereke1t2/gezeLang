Documentation

Welcome to the AmharicScript documentation! This guides you through writing valid Python using Amharic syntax.

Why AmharicScript?

Designed for Ethiopian students learning programming for the first time, lowering the barrier to entry by using their native tongue. Generated code has ZERO overhead and runs perfectly anywhere Python runs.

Structure

  • Getting Started: Setup and basic CLI usage
  • Keywords: Searchable dictionary of all Python to Amharic mappings
  • Adding a Language: Guide for providing a translation JSON to natively support Oromo, Tigrinya, etc.
