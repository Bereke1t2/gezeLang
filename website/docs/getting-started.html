Getting Started

1. Installation

You can install AmharicScript globally via pip:

pip install amharicscript

Or clone the source code to start modifying or adding languages:

git clone https://github.com/Bereke1t2/gezeLang.git
cd amharicscript

2. Your First Program

Create a file named hello.amh and add the following:

አሳይ('ሰላም ዓለም!')

3. Running Files (CLI)

Command Action
python translator.py program.amh Transpile + run the program
python translator.py program.amh --emit Emit program.py file only
python translator.py program.amh --lang oromo Use Oromo language adapter
python translator.py --check program.amh Syntax check only
python translator.py --list-langs List all available adapters
python translator.py --stats program.amh View language usage statistics
python translator.py --compare program.amh amharic oromo Compare multiple adapters

4. VS Code Extension Setup

The recommended way to write AmharicScript is with our official VS Code extension which brings:

  • Syntax highlighting
  • Hover popup translation / documentation
  • Snippets

Search for AmharicScript in the sidebar, or compile it manually from the vscode-extension folder using vsce package.
